## Getting Started

### Build Typings

```
npm run build
```

### Validate Spec

```
npm test
```

### Download and Access via npm

#### Install

```bash
npm install --save @addresszen/openapi
```

#### Require specs

JSON and YAML specs can be accessed at `node_modules/@addresszen/openapi/dist/openapi.[json|yaml]`
